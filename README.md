# My-Termux-Dotfile
My termux dotfile
### Install
Install git using
````
apt install git -y
````
clone this repo using 
````
git clone https://github.com/Begitdj/My-Termux-Cfg.git
````
go in the folder using 
````
cd My-Termux-Cfg
````
start using 
````
bash install.sh
````

# My-Termux-Dotfile
My termux dotfile
### Install
First update repository ans upgrade package using
````
apt update && apt upgrade
````
Install git using
````
apt install git -y
````
clone this repo using 
````
git clone https://github.com/Begitdj/My-Termux-Dotfile.git
````
go in the folder using 
````
cd My-Termux-Cfg
````
install using 
````
bash install.sh
````
